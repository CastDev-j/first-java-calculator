# Primera Calculadora

Calculadora básica desarrollada en Java con Maven y JavaFX.

## Características
- Operaciones básicas: suma, resta, multiplicación, división
- Interfaz gráfica con JavaFX
- Botón Clear para limpiar

```bash
mvn clean compile
mvn javafx:run
```

## Requisitos
- Java 11+
- Maven 3.6+
